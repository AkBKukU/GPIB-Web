## Install ##

### GPIB ###
 - [Linux GPIB Driver](https://sourceforge.net/projects/linux-gpib/)
 - For NI-USB-HS: `modprobe ni_usb_gpib`
 - Python interface included with driver

### Web ###
 - `pip install flask`


## Example ##

Run example code with:

    flask --app web-example run --host=0.0.0.0
